# Hotel Room Booking System

A Java-based hotel room booking system with SQL backend for managing rooms, reservations, and customers. This project demonstrates full-stack concepts, database integration, and object-oriented programming in a real-world scenario.

## Features
- Add, update, and delete hotel rooms.
- Book and cancel reservations.
- Search for available rooms by date and type.
- Manage customer information.
- View booking history and reports.
- Input validation and error handling.

## Technologies
- **Programming Language:** Java (JDK 8+)
- **Database:** MySQL / MSSQL
- **Database Connectivity:** JDBC
- **IDE:** Eclipse / IntelliJ IDEA / Visual Studio Code
- **Build Tool:** Optional (Maven / Gradle)
